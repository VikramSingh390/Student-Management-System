# Student Management System

A desktop-based Student Management System built using Python Tkinter and MySQL.

## Features
- Add Student
- Search Student
- Update Student
- Delete Student
- Show Student
- Export Data to CSV

## Technologies Used
- Python
- Tkinter
- MySQL
- PyMySQL

## How to Run
1. Install required modules:
   pip install ttkthemes pymysql

2. Start MySQL server

3. Run the project:
   python main.py!
